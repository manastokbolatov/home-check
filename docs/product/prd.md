# Product Requirements Document

# HomeCheck MVP

Version: 1.0

---

# Product Overview

HomeCheck is a mobile application that helps caregivers ensure their dependents leave home safely by using simple daily checklists and timely notifications.

The application is designed to reduce everyday anxiety through clear communication rather than continuous monitoring.

---

# Goals

The MVP aims to:

- Help dependents complete important tasks before leaving home.
- Give caregivers confidence that everything is okay.
- Notify caregivers when attention may be needed.
- Keep the experience simple and fast.

---

# User Roles

## Caregiver

Can:

- Create a family
- Invite dependents
- Create and edit checklists
- View dependent status
- Receive notifications

---

## Dependent

Can:

- Join a family
- View personal checklist
- Complete tasks
- Confirm leaving home

---

# MVP Features

## Authentication

- Email login
- Google login

---

## Family Management

- Create family
- Join family using invitation code

---

## Checklist Management

Caregivers can:

- Create checklist
- Edit checklist
- Delete checklist

Each dependent owns one personal checklist.

---

## Daily Routine

Dependent:

1. Opens HomeCheck
2. Completes checklist
3. Presses "Leave Home"

---

## Notifications

Caregiver receives notifications when:

- Dependent confirms leaving.
- No confirmation is received within the configured time.
- Dependent leaves with unfinished tasks.

---

# Non-Goals

The MVP does NOT include:

- GPS tracking
- Automatic location detection
- AI recommendations
- Voice assistant
- Smart home integration
- Statistics
- History

---

# Success Metrics

The MVP is successful if users can:

- Complete a checklist in under one minute.
- Understand family status within a few seconds.
- Receive reliable notifications.

---

# Product Principles

- Simplicity first.
- Warn, never block.
- One clear purpose per screen.
- Focus on peace of mind.
